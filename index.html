RefreshJS

UX developers, you don't need to reload your page anymore!

RefreshJS is a nodejs server and a chrome extension which watches your css, images, javascript, fonts folders for changes, and updates the page automatically whenever any file is updated.

Get it now!
  • Download git and nodejs & npm.
  • git clone https://github.com/prajwalit/RefreshJS.git
  • cd RefreshJS
  • npm install connect
  • node refresh.js
  • Go to: http://localhost:7725/
  • Install chrome extension.
  • Configure!
